# frontend-c (user panel)

React components for admin user creation flows, consumed by `frontend-a`. Storybook documents the library. The Docker image serves the Storybook static build.

| Area  | Notes                                     |
| ----- | ----------------------------------------- |
| Build | Vite library mode + `vite-plugin-dts`     |
| Docs  | Storybook 8 + `@storybook/react-vite`     |
| Tests | Vitest + Testing Library, 80% threshold |
